# Machine Learning

Predictive analytics for manufacturing time estimation based on historical performance data.

## Planned Features

### Hours Prediction
- Predict total hours per work order based on:
  - Part count and complexity
  - Material types
  - Processing station requirements
  - Historical performance data

### Station Time Estimation
- Per-station processing time predictions:
  - Nesting station hours
  - Panel saw hours
  - Edgebanding hours
  - P2P (Point-to-Point) hours
  - Miter cut hours

### Performance Analysis
- Trend analysis over time
- Efficiency metrics by operator/shift
- Bottleneck identification

### Resource Planning
- Capacity planning
- Schedule optimization
- Resource allocation recommendations

## Data Sources

The ML models will use data from:
- `db_migration/` - Historical production data
- `webapp/` - Real-time metrics and tracking
